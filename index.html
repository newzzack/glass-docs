1. Glass文档

1.1. 参与贡献

克隆master分支代码

git clone git@github.com:Rokid/glass-docs.git

1.2. 本地验证

  • 安装 node.js 环境
  • 安装 gitbook 包
    npm install -g gitbook-cli
  • 初始化 gitbook
    gitbook install
  • 本地运行 gitbook
    gitbook serve
  • 浏览器打开
    http://localhost:4000/

1.3. 发布

  • 初始化 npm 项目
    npm install
  • 发布项目
    npm run publish
